# Login Application

## Overview

This is a simple React application for user authentication using a dummy authentication service. The app includes a login form, responsive design, and integrates with the dummy authentication API.

## Features

- **Authentication**: Uses `https://dummyjson.com/auth/login` for user authentication.
- **Responsive Design**: Adapts to both mobile and desktop screens.
- **Font**: Roboto.
- **Consistent Sizing**: All dimensions are multiples of 4px.
